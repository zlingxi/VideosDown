写一个YouTube、vmieo视频下载器
1. 使用python语言以及pyside6作为GUI，不使用import导入yt-dlp模块，而是使用当前目录的resource文件夹下的yt-dlp.exe以及ffmpeg.exe文件处理下载和合并视频音频流为MP4文件
2. 第一步，初始化启动的时候，弹窗提示输入ID，检测用户ID是否有时间授权，有的话创建本地文件，下次就不用验证了，并且打开软件，授权通过连接https请求访问服务器里面的.env文件获取用户ID和时间，也帮我写一下env文件；
3. 第二步，点击一个命名为“检测代理”按钮，检测代理是否可以连接，可以的话允许用户输入YouTube或者vmieo链接，不可以就弹窗提示用户代理不可用，更换代理等字样提示
4. 第三步，解析地址后，获取可以下载的分辨率，使用下拉列表显示
5. 第四步，点击下拉列表后，选择合适的分辨率后，选择一个保存的路径
6. 第五步。点击下载，如果上面没有检测到分辨率或者没有输入链接，就提示一下用户操作，日志提示下载进度，最后显示下载完成
