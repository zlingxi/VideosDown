最初做 VideosDown，只是出于一点热爱。总想着利用下班和闲暇的碎片时间，熬夜打磨这些小功能，无偿分享出来，能帮到大家、被大家使用，就觉得很有成就感，常常不知不觉就忙到深夜。
一直以来都是纯粹靠热爱在坚持，没有任何收益支撑也从未在意。可慢慢发现，不少朋友在免费使用的同时，不仅缺少支持与体谅，还会提出各类严苛的功能要求，甚至随意指点评判，长期下来消耗了太多精力与热情。
后来慢慢想明白，单纯靠热爱很难长久维系持续的开发与维护。因此决定调整模式，转为付费使用。付费之后，大家的反馈也更真诚理性，我也能更专注地投入时间，及时修复问题、迭代新功能，把工具做得更完善稳定。

喜欢的话就支持一下哦~~~
